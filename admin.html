Admin Reference

Full command reference for Asmodeus and authorized admins. All commands require an ID in the ADMIN_IDS whitelist.

admin items

descriptionrequiredShown in shop and purchase confirmation

quantityrequiredTotal stock available
categoryrequiredUsed for grouping in the shop

fulfillmentoptionalauto (default) or manual
